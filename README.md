# Django-проект интернет-магазина

## Приложение catalog
В проекте реализовано приложение под названием catalog.

Приложение содержит следующие шаблоны:

- HTML-шаблон home.html для домашней страницы;
- HTML-шаблон contacts.html для страницы с контактной информацией.

## Использование:

1. Клонируйте репозиторий:

```
https://github.com/Sermin22/DjangoProjectShop
```
2. Перейдите в ветку для разработки <develop>.
